# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview
This project builds a machine learning model to classify iris flowers into three species:
- Setosa
- Versicolor
- Virginica

The classification is based on four features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🎯 Objective
To develop a supervised machine learning model that accurately predicts the species of an iris flower using its physical measurements.

---

## 📊 Dataset Information
- Total Samples: 150
- Features: 4 numerical features
- Target Variable: Species (3 classes)
- Dataset is clean with no missing values
- Balanced dataset (50 samples per class)

Dataset Source: UCI Machine Learning Repository / Seaborn Library

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🔄 Workflow

---

## 📈 Exploratory Data Analysis
- Pairplot shows clear separation of Setosa species
- Versicolor and Virginica have slight overlap
- Petal features are highly correlated and most important

---

## 🤖 Model Used
- Logistic Regression

---

## 📊 Model Performance
- Accuracy: ~95% to 100%
- Confusion Matrix shows minimal misclassification
- Model performs well across all three classes

---

## 🔍 Key Insights
- Petal length and petal width are the most influential features
- Dataset is perfectly balanced, improving model reliability
- Logistic Regression performs efficiently on this dataset
